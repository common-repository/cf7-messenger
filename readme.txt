Cf7 Messenger will help you send the Contact Form 7 data to your Facebook Messenger immediately.

== Description ==

When someone send the message for you via Contact Form 7 on your website. Cf7 Messenger will call to our API server and send this message to the Facebook account you've connected.
Our API DON'T collect any infomation from you, just forward the message from Contact Form 7 to your Facebook account via Messenger. If you still worry about the infomation in message you can choose the option to just send the notification without the real message content.

Let's see how the plugin works in this video

You'll get the notification on all your devices immediately. Therefore, you can contact your customer as quickly as possible.

= Features =

- Send the Contact Form 7 data to your Facebook Messenger immediately
- Unlimited message
- Get notification in all your devices

== Installation ==

1. Unzip the cf7-messenger.zip
2. Copy cf7-messenger folder to wp-content/plugins
3. Go to Plugins > Installed Plugins, find Cf7 Messenger and click Active
4. Go to Settings > Cf7 Messenger and fill the Connect Code
5. Enjoy!
